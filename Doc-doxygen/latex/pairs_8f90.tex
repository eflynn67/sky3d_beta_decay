\item 
module \mbox{\hyperlink{namespacepairs}{pairs}}
\begin{DoxyCompactList}\small\item\em The principal part of this module is the subroutine {\ttfamily pair}, which computes the pairing solution based on the B\+CS model. It is the only public part of this module. The other subroutines are helper routines that directly use the single-\/particle properties defined in module {\ttfamily Levels}. The module variable {\ttfamily iq} controls whether the solution is sought for neutrons ({\ttfamily  iq=1 } or protons {\ttfamily  iq=2 } and accordingly the single-\/particle levels from {\ttfamily  npmin(iq) } to {\ttfamily  npsi(iq) } are affected. \end{DoxyCompactList}\end{DoxyCompactItemize}
