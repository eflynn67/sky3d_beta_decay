complex(db), dimension(:,:,:,:), intent(inout)  pout,
integer, intent(in)  ifadd 
)

The routine does a matrix multiplication of a real matrix with a complex wave function along the x-direction. ifadd allows accumulating results but is not used in the present code.

The operation carried out here includes a loop over the spin index.

Parameters
[in]xmatREAL(db), takes the matrix dimensioned as a square matrix with the number of points in the x-direction.
[in]pinnCOMPLEX(db), array, takes the input wave function.
[out]poutCOMPLEX(db), array, returns the multiplied wave function.
[in]ifaddINTEGER, if nonzero, the result of the multiplication is added to the input wave function.
