neutron and proton start and ending indices

The neutron states are numbered npmin(1) through npsi(1) and the proton states run from npmin(2) through npsi(2). Protons follow neutrons, so npmin(1)=1 and npmin(2)=npsi(1)+1. Note that for each particle type the number of states can be larger than the particle number, as states may be fractionally occupied or even empty. If initialization is not from fragments, npsi(2) as an input value refers to the total number of proton states, it is later updated (in init.f90) to its normal meaning as the final index for proton states, which coincides with the total number of states, npsi(2)=nstmax.

◆ alloc_levels()

subroutine levels::alloc_levels

This subroutine allocates all the arrays associated with single-particle wave functions.

Note that while most have dimension nstmax, psi itself is dimensioned for the number nstloc of wave functions on one specific processor. It also records the isospin value.
